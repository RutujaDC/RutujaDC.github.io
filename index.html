About Me

Hello! I'm Rutuja D Chikkorde, a passionate developer with a focus on solving real-world problems through code. This portfolio showcases my solutions to complex challenges, specifically targeting AWS-related problems using advanced Data Structures and Algorithms (DSA). I leverage dynamic programming for problems like Longest Common Subsequence and Subset Sum, explore various tree structures like AVL and Red Black Trees, and tackle mathematical concepts such as Fermat's Little Theorem and Catalan Numbers. Additionally, I implement advanced algorithms including A* Search, Kadane's Algorithm, and Segment Trees. My portfolio also covers essential graph algorithms and principles like Lazy Propagation and Sliding Window. Each solution is accompanied by sample code and explanations, demonstrating how I apply these concepts to create efficient and scalable solutions.
